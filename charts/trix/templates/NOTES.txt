trix has been installed!

1. Check the deployment status:
   kubectl get pods -l "app.kubernetes.io/name={{ include "trix.name" . }}" -n {{ .Release.Namespace }}

2. View the logs:
   kubectl logs -l "app.kubernetes.io/name={{ include "trix.name" . }}" -n {{ .Release.Namespace }} -f

3. Check vulnerability statistics:
   kubectl exec -n {{ .Release.Namespace }} {{ include "trix.postgresql.fullname" . }}-0 -- \
     psql -U {{ .Values.postgresql.user | default "trix" }} -c \
     "SELECT state, severity, COUNT(*) FROM vulnerabilities GROUP BY state, severity"
